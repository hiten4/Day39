# Day 39: NLP Patterns & Aspect-Level Sentiment

This directory contains the notebook and scripts for the Day 39: NLP Patterns & Aspect-Level Sentiment assignment.

## Environment & Dependencies
- **Python Version:** 3.9+
- **Required Packages:**
  ```bash
  pip install pandas spacy textblob vaderSentiment
  python -m spacy download en_core_web_sm
  ```


## How to Run
1. Open up a terminal in this directory.
2. Ensure you have activated your preferred virtual environment.
3. Install the dependencies listed above.
4. Launch Jupyter Notebook or VS Code:
   ```bash
   jupyter notebook PART-A-B-39.ipynb
   ```
5. Click **Run All** in the notebook.

## Expected Output
Running the notebook end-to-end will produce:
- Evaluations of 5 complex NLP patterns (sarcasm, negation, code-mixing) and extracted aspect-sentiment pairs.
